Por que cursar SI1?

Códigos de fácil compreensão, conhecimento em tecnologia Web e experiência em desenvolvimento de sistemas são aspectos vistos com bons olhos por empresas de todo o mundo, e a disciplina de SI1 pode ajudá-lo com isso.

Qualidade de Código

Supondo que houvesse a necessidade de alterar um trecho de código do software que você desenvolveu na disciplina de LP2, afim de aprimorá-lo, o quão difícil isso seria para você? Na disciplina de Sistemas de Informações I, você aprenderá que o segredo do desenvolvimento de um software robusto, porém fácil de se evoluir, está na qualidade do código implementado.

Desenvolvimento Web

Afim de capacitar o aluno cada vez mais para o que o mercado de trabalho exige, a disciplina de SI1 é um pontapé para aperfeiçoar os conhecimentos em desenvolvimento web, algo bastante cobrado pelas empresas.

Experiência

Como diz o antigo ditado, toda experiência é válida. Em SI1, o aluno terá a oportunidade de desenvolver um projeto utilizando diversas tecnologias web e ferramentas de controle de versão.
