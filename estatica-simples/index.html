Por que cursar SI1?

Qualidade de Código

Supondo que houvesse a necessidade de alterar um trecho de código do software que você desenvolveu na disciplina de LP2, afim de aprimorá-lo, o quão difícil isso seria para você? Na sua opinião, se uma outra pessoa fosse o responsável por trabalhar na evolução do sistema, ela compreenderia o código facilmente? Muitos acreditam que aumentando o número de desenvolvedores do sistema, a prdutividade se tornaria maior. No entanto, na disciplina de Sisttemas de Informações I, você aprenderá que o segredo do desenvolvimento de um software robusto, porém fácil de se evoluir, está na qualidade do código implementado.

Desenvolvimento Web

Afim de capacitar o aluno cada vez mais para o que o mercado de trabalho exige, a disciplina de SI1 é um pontapé para aperfeiçoar os conhecimentos em desenvolvimento web, algo bastante cobrado pelas empresas.

Experiência

Como diz o antigo ditado, toda experiência é vàlida. Em SI1, o aluno terá a oportunidade de desenvolver um projeto utilizando diversas tecnologias web e ferramentas de controle de versão.
